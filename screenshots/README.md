## How to Take Screenshots

### Android
1. Run the app on Android emulator or device
2. Press `Ctrl + S` (Windows/Linux) or `Cmd + S` (Mac) in the emulator
3. Or use the camera button in the emulator toolbar

### iOS
1. Run the app on iOS simulator or device
2. Press `Cmd + S` in the simulator
3. Or use `Cmd + Shift + 4` on Mac and click on the simulator

### Recommended Size
- Resolution: 1080x1920 (portrait) or scaled proportionally
- Format: PNG for better quality
- File size: Keep under 500KB per image
